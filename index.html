    Powered by OpenAI Dall-E 3 & Dall-E 2, Roominate is a tool to help you design your home. I developed this app with four other engineers, focusing on the frontend and UX/UI. Features full CRUD functionality for projects and comments. Roominate uses MongoDB, Mongoose, Express, JavaScript, React and Redux.

    Burnr

    Burnr, a clone of Flickr, is a photo-sharing platform and social network where users can share and comment on each other's photos from Burning Man. Burnr allows for a user to upload, view, remove and edit photos and comments. Burnr was developed using Ruby on Rails, PostgreSQL, JavaScript, React, and Redux.

    Go to Burnr ReadMe
